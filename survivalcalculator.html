The LoPucki-Doherty Bankruptcy Survival Calculator: Prediction

Enter any combination of these eleven variables, click on “Calculate,” and the program predicts the likelihood that the debtor company will continue in business as a stand-alone company beyond the end of the bankruptcy case.

Example      Input
No    Enter “yes” if the debtor publicly indicated an intent to sell the businesses, “no” if not.
4.1    Enter the prime rate of interest one year prior to filing.
0    Enter 2 if the case is prepackaged, 1 if prenegotiated, 0 if neither
-.08    Enter the ratio of the debtors equity to assets before bankruptcy
0    Enter 1 if the debtor’s primary business is manufacturing, 0 if not.
21    Enter the distance from debtor’s headquarters city to the local bankruptcy court city
1    Enter 1 if a creditors committee was appointed in first 24 days of the case, 0 if not.
0    Enter 1 if the debtor’s Earnings Before Interest and Taxes was greater than zero, 0 if not.
7    Enter the number of prior BRD cases in which the judge was the judge at confirmation.
845,653,000    Enter the debtor’s assets before bankruptcy, in current (inflation adjusted) dollars.
1    Enter 1 if the court approves a DIP loan, 0 if not.


 Output
 Probability the debtor company will survive bankruptcy

The variables for this model were selected based on the regression model in table __ of Lynn M. LoPucki and Joseph W. Doherty, Bankruptcy Survival (July 28, 2014). Data used in the calculations are updated monthly.
